## ðŸ” Authentication System

### Flow
1. User visits landing page (logged out)
2. Clicks "Login" button
3. Enters credentials or clicks "Demo Login"
4. Form validates input
5. Creates user object
6. Sets `isLoggedIn = true`
7. Navbar updates to show profile button
8. Protected pages become accessible

### Protected Routes
- /profile (redirects to /login if not authenticated)
- /activity (redirects to /login if not authenticated)
- /history (redirects to /login if not authenticated)

### State Management
- `isLoggedIn`: boolean
- `currentUser`: { name, email, avatar, joinDate, role }
- Passed via props to components
- Updated via setters from LoginPage
